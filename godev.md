Este script permite hacer una búsqueda en un directorio donde están todos los proyectos de desarrollo de software
Busca un directorio que contenga un patrón de texto indicado, si existe un solo directorio que obedezca a este patrón cambia del directorio al mencionado directorio
Si existe más de un directorio que obedezcan a este patrón de texto, hace una lista de los directorios existentes
Si no existe ningún directorio que corresponda al patrón indicado muestra la leyenda con menos F para forzar la creación de un directorio con ese nombre adentro del directorio de base donde se encuentran todos los proyectos de desarrollo de software. Cada uno en un directorio.
